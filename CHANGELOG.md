# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-21

### Added

- `deepEqual()` for comparing objects, arrays, Maps, Sets, Dates, RegExps, and TypedArrays
- Strict mode with `Object.is` semantics (NaN === NaN, -0 !== +0)
- Circular reference detection via `circular` option
